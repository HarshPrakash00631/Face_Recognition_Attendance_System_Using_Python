# Face_Recognition_Attendance_System_Using_Python
# 🎯 Face Recognition Based Attendance System

## 📌 Overview
This project is a **Face Recognition Based Attendance System** built using **Python, OpenCV, and Tkinter**.  
It captures face images, trains a recognition model, and automatically marks attendance using a webcam.

---

## 🚀 Features
- 👤 Register new users (ID & Name)
- 📸 Capture face images
- 🧠 Train model using LBPH algorithm
- 🎯 Real-time face recognition
- 📝 Automatic attendance marking
- 📊 Attendance saved in Excel format
- 🔒 Password protection for training
- 🗣️ Voice feedback for attendance

---

## 🛠️ Technologies Used
- Python  
- OpenCV  
- Tkinter  
- NumPy  
- Pandas  
- Pillow (PIL)  
- pyttsx3  
- OpenPyXL
